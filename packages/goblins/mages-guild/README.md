# @goblinos/mages-guild

Mages Guild CLI keeps quality gates tight and validates the knowledge vault.

## Usage

```bash
pnpm -C GoblinOS mages-guild --help
```

### Commands
- `quality:lint` — run the monorepo lint and security umbrella.
- `quality:full` — alias for `quality:lint` to align with dashboards.
- `vault:validate` — validate the Forge knowledge vault.
- `tools` — list Mages guild toolbelt assignments.
- `roadmap` — view upcoming anomaly automation milestones.
