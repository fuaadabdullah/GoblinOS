# @goblinos/huntress-guild

Huntress Guild CLI drives smoke probes and signal-scouting workflows.

## Usage

```bash
pnpm -C GoblinOS huntress-guild --help
```

### Commands
- `smoke` — execute the Kubecost smoke probe.
- `signals:plan` — preview upcoming automation milestones.
- `tools` — list Huntress guild toolbelt assignments.
