# @goblinos/keepers-guild

Keepers Guild CLI automates secrets, compliance, and storage hygiene.

## Usage

```bash
pnpm -C GoblinOS keepers-guild --help
```

### Commands
- `secrets:audit` — audit API key documentation and .env hygiene.
- `security:scan` — run security compliance checks.
- `secrets:playbook` — open the secrets management playbook.
- `storage:cleanup` — run the weekly storage cleanup routine.
- `storage:consolidate` — execute disk consolidation.
- `storage:space-saver` — archive heavy caches and venvs.
- `system:clean` — perform system-level cache purge.
- `tools` — list Keepers guild toolbelt assignments.
